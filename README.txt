[Ambassador](https://www.getambassador.com) helps you turn your customers into brand ambassadors. This plugin adds the Ambassador universal snippet to your WordPress site.

== Installation ==

1. Activate the plugin through the 'Plugins' menu in WordPress
2. Go to the snippet page in your Ambassador account
3. Copy your Universal ID
4. Go to the WordPress settings page for the Ambassador plugin
5. Paste and save the Universal ID

== Frequently Asked Questions ==

= Where do I find my Universal ID =

The Universal ID can be found when logged into your Ambassador account [here](https://getambassador.com/v3/c/build/snippet/).
